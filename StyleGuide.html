P02 - Consolidated Pattern Library

This page is a style guide for the beer store called Well-being Beer (WbB). This Styleguide is intended to show the interactive elements, text styling, modular pattern elements, and general styling of the site.

On this page:

Interactive Element Text Element Modular Element

INTERACTIVE ELEMENTS

In-Text Link

Any more question? Please contact us for further information!

Description:

text link will be used to lead audiences for more information. The links are going to be highlighted into yellow text

Designed by:

Chae Yeon Park and inspired from Paul's Tutorial

CSS Code:

.intextlink

Sample Code:

<p>Any more question? Please <a class ="intextlink" href="#contact">contact us</a> for further information!</p>

Submit/Reset Buttons

Description:

Same as Action Buttons, will apply on triggering actions in the site. Will be apply on Text input sections such as contact or review comments section.

Designed by:

Chae Yeon Park and inspired from W3School.com

CSS Code:

.button .button1 .button2

Sample Code:

<button class ="button button1">SUBMIT</button> <button class = "button button2">RESET</button>
