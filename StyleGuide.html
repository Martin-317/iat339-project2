P02 - Consolidated Pattern Library

This page is a style guide for the beer store called Well-being Beer (WbB). This Styleguide is intended to show the interactive elements, text styling, modular pattern elements, and general styling of the site.

On this page:

Interactive Element Text Element Modular Element

INTERACTIVE ELEMENTS

In-Text Link

Any more question? Please contact us for further information!

Description:

text link will be used to lead audiences for more information. The links are going to be highlighted into yellow text

CSS Code:

.intextlink

Sample Code:

<p>Any more question? Please <a class ="intextlink" href="#contact">contact us</a> for further information!</p>

Buttons

Description:

Buttons will apply on triggering actions in the site. Will be apply on Text input sections such as contact or review comments section.

CSS Code:

.button .brown_btn .yellow_btn .sizebtn

Sample Code:

<button class ="button brown_btn">SUBMIT</button> <button class = "button yellow_btn">RESET</button> <button class = "sizebtn">12 oz CAN</button>

Text Input Field


Description:

Will be apply on text submission on the e-commerce such as Contact modules or Purchase module.

CSS Code:

.input

Sample Code:

<label>Name:</label> <input type="text" class="input" placeholder="First Name"> <input type="text" class="input" placeholder="Last Name">

TEXT ELEMENTS

Headers

MAIN HEADER

SUB HEADER 1

SUB HEADER 2

SUB HEADER 3

Description

Each header has different purpose. Main Header is for main title for pages, Sub Header 1 and 2 are titles for each sub-contents. And lastly, Sub Header 3 is for titles for products in product lists.

CSS Code:

.mainheader

Sample Code:

<h1>MAIN HEADER</h1> <h2>SUB HEADER 1</h2> <h3>SUB HEADER 2</h3> <h4>SUB HEADER 3</h4>

Paragraph Sample

Looking for where to find healthier and fresh craft beer releases? You can find products in WbB including a wider selection of products with less sugar, calories, carbs — even low and no-alcohol options to suit every lifestyle!

stylized with regular Work Sans fonts and dark brown color (#3A2E2A)

.para

<p>Looking for where to find healthier and fresh craft beer releases? You can find products in <strong>WbB </strong> including a wider selection of products with less sugar, calories, carbs — even low and no-alcohol option s to suit every lifestyle! </p>

Quick Add Module

Description:

This module pattern will be displayed when users click "Quick Add" button on product list. It is for instantly add products into cart without vising product detail pages.

CSS Code:

.quickadd .label2 .select .quanti .button .button2
